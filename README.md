### About

**Первый самостоятельно свёрстанный макет (pure CSS)**
Первая попытка на пути к грамотной архитектуре и пониманию

### How to run

1. Клонировать репозиторий.
2. Установить зависимости с помощью команды **p/npm install**.
3. Прописать в терминале **p/npm run dev**.
4. Кликнуть по ссылке с **localhost** с активным **Ctrl** или написать адрес в строке браузера.
